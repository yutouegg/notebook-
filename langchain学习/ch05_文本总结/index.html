    ch05文本总结

    使用langchain对文本进行摘要

    1.仅用提示词进行摘要：

    from dotenv import load_dotenv
from langchain_openai import ChatOpenAi
import os

load_dotenv()
openai_key = os.getenv("OPENAI_API_KEY")
llm = ChatOpenAI(openai_api_key=openai_api_key,model_name='gpt-4o')

    创建提示词：

    from langchain.schema import(
    AIMessage,
    HumanMessage,
    SystemMessage
)

#定义文章
speech="""
人工智能与深度学习：前沿技术的发展

近年来，人工智能（AI）和深度学习技术在各个领域取得了显著的进展。深度学习是机器学习的一个子领域，它使用多层神经网络来学习数据的复杂模式。这些技术已经被广泛应用于图像识别、自然语言处理、自动驾驶等领域。

深度学习的基础
深度学习的核心在于神经网络模型，这些模型由多个层组成，每一层都试图学习输入数据的不同特征。常见的神经网络包括卷积神经网络（CNN）和循环神经网络（RNN），前者常用于图像处理，后者则在处理序列数据（如文本）方面表现优异。

应用与挑战
人工智能的应用范围非常广泛，从智能语音助手到医疗诊断，无处不在。然而，尽管取得了许多进展，人工智能技术仍面临许多挑战，例如模型的透明度、数据隐私问题，以及在真实世界场景中的适应性。

未来展望
随着计算能力的提升和数据量的增加，深度学习技术有望在更多领域发挥作用。研究人员正在探索更高效的模型和算法，以解决当前的技术瓶颈，并进一步推进人工智能的发展。
"""

chat_messages=[
    SystemMessage(content='你是一个专业的助手擅长对文章进行总结'),
    HumanMessage(content=f'请对下面的文章进行简短的摘要总结:\n TEXT: {speech}')
]

print(llm.invoke(chat_messages).content)

    运行结果如下：
    这篇文章概述了人工智能和深度学习的基本概念、应用领域和未来发展方向。如果你有兴趣深入了解，可以查阅相关的专业书籍和学术论文。

    2.使用Prompt Templates：

    要构建通用的文章摘要系统，提示词还是首选Prompt Templates

    from langchain.chains import LLMChain
from langchain import PromptTemplate

generic_template='''
对下面的文章进行总结:
Speech : `{speech}`
将总结翻译成下方的语言 {language}.

'''
prompt=PromptTemplate(
    input_variables=['speech','language'],
    template=generic_template
)

print(prompt.format(speech=speech,language='English'))

    3.使用summarize_chain进行总结：

    from PyPDF2 import PdfReader

pdfreader = PdfReader(pdf_path)

text = ''
#将每一页的文字转成字符串格式
for i, page in enumerate(pdfreader.pages):
    content = page.extract_text()
    if content:
        text += content

from langchain.docstore.document import Document
docs = [Document(page_content=text)]

    将字符串格式的文本转成document格式主要是因为：
    langchin通过统一的接口处理各种文档，即是document。这种一致性使得系统更易于扩展和维护。
     Document 对象通常包含元数据，如文档的标题、作者、发布日期等。这些信息有助于更好地理解和处理文本内容。 

    

template = '''请对下面的文章进行简短的摘要总结:
Speech: `{text}`
'''
prompt = PromptTemplate(
    input_variables=['text'],
    template=template
)

from langchain.chains.summarize import load_summarize_chain

chain = load_summarize_chain(
    llm,
    chain_type='stuff',
    prompt=prompt,
    verbose=False
)
output_summary = chain.invoke(docs)

    4.Map Reduce对长文本进行总结：

    将文本进行分词逐个总结： 

    from langchain.text_splitter import RecursiveCharacterTextSplitter
text_splitter = RecursiveCharacterTextSplitter(chunk_size=1000, chunk_overlap=20)
chunks = text_splitter.create_documents([text])

chain = load_summarize_chain(
    llm,
    chain_type='map_reduce',
    verbose=False
)
summary = chain.run(chunks)

    这个文档的摘要质量与分段大小和重叠大小的选择也有很大的关系
