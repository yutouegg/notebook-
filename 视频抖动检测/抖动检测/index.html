    视频抖动检测

    视频抖动情况检测：

    1.检测原理：

    使用光流法（Optical Flow）来检测帧间的运动。计算每帧的平均运动幅度，并将其标准化。 根据设定的阈值，识别出抖动的帧段。 将连续的抖动帧段（持续时间超过设定的最小时长）标记为抖动片段。 返回一个包含所有检测到的抖动片段信息的列表，每个片段包括开始帧、结束帧、特征ID、描述和使用的算法名称。

    def process_video2(file_path: str, threshold: float = 0.0, logger=None):
    #打开视频文件，读取视频相关的信息（帧率和总帧数）
    cap = cv2.VideoCapture(file_path)
    fps = cap.get(cv2.CAP_PROP_FPS)
    total_frames = int(cap.get(cv2.CAP_PROP_FRAME_COUNT))
    #读取第一帧，如果不存在则报错
    ret, old_frame = cap.read()
    if not ret:
        raise ValueError("Failed to read the video file")

    #转为灰度图，减少了计算量。掩码的作用也是在一个框中检测，同样减少计算量
    old_gray = cv2.GaussianBlur(cv2.equalizeHist(cv2.cvtColor(old_frame, cv2.COLOR_BGR2GRAY)), (5, 5), 0)
    mask = np.zeros_like(old_gray)
    cv2.rectangle(mask, (100, 100), (300, 300), 255, -1)

    feature_params = dict(maxCorners=10000, qualityLevel=0.05, minDistance=5, blockSize=5)
