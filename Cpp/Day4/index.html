    1.判断本机大端or小端编程：

    1.代码思路：

    定义一个整型0x12345678，转换成char类型，整型四字节，char型取第一个字节打印，如果是18，则是大端，如果是120则是小端。

    2.代码：

        int a = 0x12345678;

    int* p = &a;
    unsigned char first = *p;
    std::cout << first << std::endl;
    std::cout << static_cast<int>(first) << std::endl;
    return 0;

    2.使用TCP完成基础socket编程：

    1.知识点：

    • 使用 <sys/socket.h>, <netinet/in.h>, <arpa/inet.h> 等头文件进行套接字编程
    • 使用 socket() 函数创建套接字：
      参数：地址族（AF_INET 用于 IPv4）、套接字类型（SOCK_STREAM 用于 TCP协议，SOCK_DGRAG用于UDP协议）。注：UDP协议没listen和acept操作。
    • 使用 struct sockaddr_in 结构体来指定地址信息。
    • 使用 listen() 函数使套接字进入监听状态，等待客户端连接。
    • 使用 accept() 函数接受客户端的连接请求。
    • 使用 read() 和 write() 函数两端进行读写操作。
    • 使用 htons() 函数将主机字节序转换为网络字节序（用于端口号）。
    • 使用 close()函数关闭套接字。
